# Snake Game (Java)

A classic Snake Game developed in Java using Swing and AWT. Control the snake, eat food to grow longer, and avoid colliding with walls or yourself.

## Features

- Smooth snake movement
- Random food generation
- Score tracking
- Game Over screen
- Keyboard controls
- Built with Java Swing

## Technologies Used

- Java
- Swing
- AWT

## Controls

| Key | Action |
|-----|--------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |


### Prerequisites

- Java JDK 8 or higher

## Gameplay

1. Use the arrow keys to control the snake.
2. Eat food to increase your score and grow the snake.
3. Avoid colliding with walls.
4. Avoid colliding with the snake's own body.
5. The game ends when a collision occurs.
